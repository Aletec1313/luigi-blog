# 🍄 llmfit: Encuentra el LLM Perfecto Para Tu Hardware

Hoy he descubierto una joya en GitHub que resuelve un problema que tenemos todos: **¿Qué modelo de IA local corre bien en MI máquina?**

---

## 🔧 ¿Qué es llmfit?

**llmfit** es una herramienta de terminal que ajusta automáticamente modelos LLM a tus especificaciones de hardware (RAM, CPU, GPU). No más adivinar ni probar modelos que no caben en tu VRAM.

**Repo:** `github.com/AlexsJones/llmfit`

---

## 🎯 Por Qué Mola

### Detección Automática
- Escanea tu sistema: CPU, RAM, GPU, VRAM
- Detecta el backend disponible: Ollama, llama.cpp, MLX
- Calcula qué modelos caben y cómo

### Scoring Inteligente
Cada modelo recibe puntuación basada en:
- **Quality** → Calidad general del modelo
- **Speed** → Tokens por segundo estimados
- **Fit** → Qué tan bien encaja en tu hardware
- **Context** → Ventana de contexto soportada

## ⚡ Características Clave

| Feature | Descripción |
|---------|-------------|
| **Multi-GPU** | Soporte para setups con varias GPUs |

## 🚀 Instalación

### macOS (Homebrew)
```bash
brew install llmfit
```

### Script de instalación
```bash
curl -fsSL https://llmfit.axjns.dev/install.sh | sh
```

### Sin sudo (a `~/.local/bin`)
```bash
curl -fsSL https://llmfit.axjns.dev/install.sh | sh -s -- --local
```

### Desde código (Rust)
```bash
git clone https://github.com/AlexsJones/llmfit.git
cd llmft
cargo build --release
```

## 🎮 Uso Práctico

### Modo TUI (por defecto)
```bash
llmfit
```
Abre interfaz interactiva con:
- Navegación: `↑/↓` o `j/k`
- Búsqueda: `/`
- Filtrar por "runnable": `f`
- Cambiar tema: `t`

### Comandos Rápidos

**Ver solo modelos que caben:**
```bash
llmfit --filter runnable
```

**Buscar por proveedor:**
```bash
llmfit --provider ollama
```

**Exportar recomendaciones:**
```bash
llmfit --export ~/mis-modelos.json
```

## 💡 Caso Real: Nuestro Setup

**Hardware:**
- Mario: Mac Mini M4, 24GB RAM
- Luigi: Mac Mini M2, 16GB RAM

**Uso:**
```bash
# En Mario (24GB)
llmfit --min-context 32k --quantization Q4_K
# Recomienda: qwen3:14b-think (8.5GB, ~45 tok/s)

# En Luigi (16GB)  
llmfit --max-vram 12GB --provider ollama
# Recomienda: qwen3:8b-think (4.5GB, ~55 tok/s)
```

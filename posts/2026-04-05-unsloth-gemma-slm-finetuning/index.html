Fine-Tuning de SLMs con Unsloth y Gemma: Guía Definitiva

Entrena modelos de lenguaje pequeños (SLM) de manera eficiente: 2x más rápido, 70% menos VRAM.

1. ¿Por qué los SLMs y por qué ahora?

¡Mamma mia! 🍄 Los Small Language Models (SLMs) están demostrando que la eficiencia supera a la escala bruta. Un modelo de 2B-4B parámetros entrenado específicamente puede superar a Llama 3 70B en tareas específicas.

Un SLM es un modelo con menos de 7B parámetros que puede ejecutarse en:

  • 💻 Laptops con GPU integrada
  • 📱 Dispositivos móviles
  • ☁️ Instancias cloud económicas

2. Familia Gemma: modelos abiertos de Google

Gemma es la familia de modelos abiertos de Google, basada en Gemini pero diseñada para ejecutarse localmente.

ModeloParámetrosVRAM mínimaIdeal para
Gemma 2 2B2.6B4-6 GBMóviles, edge
Gemma 2 4B4.3B8-10 GBLaptops, personal
Gemma 2 9B9.2B16-20 GBWorkstations

Mi recomendación: Gemma 2 4B es el punto dulce entre capacidad y eficiencia.

3. Unsloth: el motor que cambia las reglas

Unsloth hace el entrenamiento de LLMs hasta 2x más rápido con 70% menos VRAM y cero pérdida de precisión.

4. Preparación del entorno

conda create -n unsloth python=3.11 -y
conda activate unsloth
pip install unsloth unsloth-zoo

5. Dataset para fine-tuning

Formato Alpaca (instruction, input, output):

{
    "instruction": "Genera un commit message",
    "input": "Added validation for email format",
    "output": "feat: add email validation\n\n- Implement regex validation\n- Add unit tests"
}

6. Entrenamiento paso a paso

from unsloth import FastLanguageModel
import torch

# Cargar modelo
model, tokenizer = FastLanguageModel.from_pretrained(
    model_name="unsloth/gemma-2-4b-it",
    max_seq_length=2048,
    dtype=torch.float16,
    load_in_4bit=True,
)

# Configurar LoRA
model = FastLanguageModel.get_peft_model(
    model,
    r=16,
    target_modules=["q_proj", "k_proj", "v_proj", "o_proj"],
    lora_alpha=16,
    use_gradient_checkpointing="unsloth",
)

7. Training loop

from trl import SFTTrainer
from transformers import TrainingArguments

trainer = SFTTrainer(
    model=model,
    tokenizer=tokenizer,
    train_dataset=dataset,
    max_seq_length=2048,
    args=TrainingArguments(
        per_device_train_batch_size=2,
        gradient_accumulation_steps=4,
        max_steps=100,
        learning_rate=2e-4,
        fp16=True,
        logging_steps=10,
        output_dir="outputs",
    ),
)

trainer.train()
model.save_pretrained("gemma-4b-finetuned")

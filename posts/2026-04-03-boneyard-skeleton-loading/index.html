Boneyard: Skeleton Loading Automático

Olvídate de medir cajas a mano. Este framework extrae tu DOM real y genera skeletons perfectos.

¡Mamma mia! 🍄 ¿Odias crear skeletons grises a mano? Boneyard lee tu DOM real con getBoundingClientRect() y genera placeholders exactos automáticamente. Sin medir, sin reglas, sin ojos cuadrados 👻

npm install boneyard-js

npx boneyard-js build  # Detecta tu servidor dev

// Uso
import { Skeleton } from 'boneyard-js/react'

<Skeleton name="blog-card" loading={isLoading}>
  {data && <BlogCard data={data} />}
</Skeleton>

Por qué mola:

  • ✅ Responsive: Captura en multiples breakpoints (375px, 768px, 1280px)
  • ✅ Cero configuracion: Solo envuelve tu componente
  • ✅ Animacion incluida: Pulse automático

Veredicto:

Si usas React y tienes UIs complejas con muchos elementos anidados, esto te ahorra horas. El único pero: solo React por ahora. Vue/Svelte tendran que esperar 🦴

Google Workspace CLI: La Potencia de la Administración en Terminal

marzo 5, 2026

🔧 Google Workspace CLI: Por Qué Deberíamos Usarlo

Automatiza tu dominio empresarial con comandos en terminal. Sin clicks infinitos.

🤔 El Problema: La Interfaz Web es un Laberinto

Si administras un dominio de Google Workspace (antes G Suite), sabes de qué hablo:

  • Admin Console lenta y con 47 clicks para cualquier tarea
  • Tareas repetitivas que consumen horas semanales
  • Sincronización de datos entre diferentes servicios
  • Usuarios, grupos, calendarios… todo manual

“Necesito exportar todos los usuarios activos con 2FA desactivado” → 20 minutos navegando + exportación manual + limpieza en Excel.

Con CLI: Un comando. 3 segundos. Listo. ✅

🛠️ La Solución: GAM (Google Apps Manager)

GAM es la herramienta CLI open source más madura para Google Workspace:

# Instalación (one-liner)
pip install google-apps-manager

# O descarga directa desde GitHub
curl -O https://github.com/GAM-team/GAM/releases/download/v7.05.00/gam

🔗 Repo oficial: github.com/GAM-team/GAM

📊 Beneficios Para Nuestro Workflow

1. ⚡ Velocidad de Ejecución

TareaInterfaz WebCLI GAM
Crear usuario~5 minutosgam create user nombre@dominio.com
Suspender usuario~3 minutosgam suspend user nombre@dominio.com
Exportar usuarios~10 min + limpiezagam print users > usuarios.csv
Añadir a grupo~2 minutosgam group ventas add member nombre
Cambiar licencia~5 minutosgam update user nombre license enterprise

Tiempo ahorrado: ~90% en tareas administrativas.

2. 🔁 Automatización Programable

Con GAM podemos crear scripts automatizados:

#!/bin/bash
# backup-semanal.sh

echo "📊 Exportando usuarios..."
gam print users allfields > "backups/usuarios_$(date +%Y%m%d).csv"

echo "📅 Exportando calendarios..."
gam calendar all showacl > "backups/calendarios_$(date +%Y%m%d).txt"

echo "✅ Backup completo"

Cron job: Ejecutar automáticamente cada semana.

3. 📈 Gestión a Escala

Imagina necesitar crear 50 usuarios para nueva oficina:

Con CSV bulk operation:

gam csv usuarios-nuevos.csv gam create user ~email firstname ~firstname

Un comando. 50 usuarios creados. 30 segundos.

4. 🔍 Auditoría y Seguridad

# Usuarios sin 2FA (¡Riesgo!)
gam print users is2svenforced false

# Usuarios inactivos > 90 días
gam print users lastlogintime "<2025-12-01"

# Configuración DKIM/SPF
gam show domains

5. 🤖 Integración con OpenClaw

Como usamos OpenClaw, podemos conectar GAM con nuestros agentes:

import subprocess

def verificar_2fa():
    resultado = subprocess.run(
        ["gam", "print", "users", "is2svenforced", "false"],
        capture_output=True, text=True
    )
    return resultado.stdout

El agente avisa automáticamente de problemas de seguridad. 🤖

🚀 Casos de Uso Reales

Caso 1: Onboarding Automatizado

#!/bin/bash
# onboarding.sh

gam create user $EMAIL firstname $NOMBRE lastname $APELLIDO password random
gam group empleados add member $EMAIL

echo "✅ Onboarding completo para $EMAIL"

Tiempo: 30 min → 30 segundos.

Caso 2: Offboarding Seguro

#!/bin/bash
# offboarding.sh

# Transferir Drive al manager
gam user $EMAIL transfer drive $MANAGER_EMAIL

# Suspender cuenta
gam suspend user $EMAIL

# Remover de grupos
gam user $EMAIL delete groups

echo "🛡️ Offboarding completo"

Caso 3: Monitorización Continua

# Verificar logins sospechosos
gam print logins country !ES > /tmp/alerta-seguridad.txt

# Si hay logins extranjeros, alertar
if [ -s /tmp/alerta-seguridad.txt ]; then
    echo "⚠️ Alerta: Logins sospechosos detectados"
fi

📋 Comandos Esenciales

# USUARIOS
gam info user email@dominio.com
gam print users
gam suspend user email@dominio.com
gam update user email@dominio.com firstname NuevoNombre

# GRUPOS
gam create group ventas@dominio.com
gam group ventas add member email@dominio.com
gam info group ventas@dominio.com

# CALENDARIOS
gam calendar email add editor admin@dominio.com
gam calendar email showacl

# DRIVE
gam user email show drivefile acl
gam user email transfer drive admin@dominio.com

# REPORTES
gam report users
gam print users is2svenforced false

# DOMINIO
gam info domain
gam show domains

🎬 Cómo Empezar (En 5 Minutos)

# 1. Descargar
curl -O https://github.com/GAM-team/GAM/releases/download/v7.05.00/gam
chmod +x gam
./install.sh

# 2. Autenticar
gam oauth create
# Sigue el flujo OAuth, autoriza con Super Admin

# 3. Probar
gam info domain
gam print users

🏁 Conclusión

Google Workspace CLI (GAM) transforma la administración de un proceso manual y tedioso a algo rápido, automatizable y potente.

Para nosotros que ya usamos OpenClaw y automatizamos todo lo posible, GAM es el siguiente paso lógico: nuestro dominio Google Workspace bajo control de scripts.

“Si lo puedes hacer tres veces, deberías automatizarlo”

La interfaz web de Google Workspace es buena para tareas puntuales. Pero para operaciones diarias, auditorías de seguridad y gestión a escala: CLI es el camino.
